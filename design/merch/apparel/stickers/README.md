# LOWEND NYC Sticker Design Specs

## Formats

### Die-Cut Stickers
- Individual shapes cut to design
- White border: 1/16" - 1/8" recommended
- Rounded corners: 1/8" radius

### Sticker Sheets
- Multiple designs per 8.5" x 11" sheet
- Cut lines as vector paths
- 0.25" margin from edges

## Sizes

| Type | Size | Use Case |
|------|------|----------|
| Small | 2" x 2" | Laptop corner, phone case |
| Medium | 3" x 3" | Water bottle, notebook |
| Large | 4" x 4" | Car, equipment case |
| Bumper | 11" x 3" | Car bumper, venue walls |

## Design Ideas

### Logo Variations
- LE icon (red on black, white on black)
- Full wordmark horizontal
- Full wordmark stacked

### Illustrated
- NYC skyline silhouette + LOWEND
- Sound wave visualization
- Venue map icons

### Event Commemorative
- Date + venue + lineup
- Limited edition numbering

## File Requirements

- Format: AI, EPS, or PDF (vector preferred)
- Die line: Separate layer named "CutContour" or "Die Line"
- Bleed: 0.125" beyond cut line
- Colors: CMYK or Pantone spot colors
