## Merch Typography

For apparel and physical goods:
- **Primary**: Bold condensed sans (Oswald or Bebas Neue)
- **Weight**: 700-900 for maximum impact
- **Tracking**: Tight to normal (0 to -0.02em)
- **All caps** for headlines on merch

## Font Files

Place licensed font files in this directory:
- `fonts/` — Desktop fonts (OTF/TTF)
- `webfonts/` — Web-optimized (WOFF2)
