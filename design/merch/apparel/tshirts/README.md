# LOWEND NYC T-Shirt Design Specs

## Standard Placement

### Front Chest (Left Chest)
- **Size**: 3.5" - 4" wide
- **Placement**: 3" down from collar, centered left chest
- **Best for**: Small logo, wordmark, icon

### Front Center (Full Chest)
- **Size**: 10" - 12" wide
- **Placement**: 3" down from collar, centered
- **Best for**: Large graphics, statements

### Back
- **Size**: 12" - 14" wide
- **Placement**: 3" down from collar, centered
- **Best for**: Large designs, event graphics, URL

## Print Methods

| Method | Best For | Colors | Min Order | File Format |
|--------|----------|--------|-----------|-------------|
| DTG (Direct to Garment) | Complex/full color | Unlimited | 1 | PNG, 300dpi, transparent BG |
| Screen Print | Bold, limited colors | 1-6 colors | 25+ | AI/EPS with spot colors |
| Heat Transfer | Small runs, photos | Unlimited | 1 | PNG, 300dpi |

## File Requirements

### DTG
- Resolution: 300 DPI minimum
- Format: PNG with transparent background
- Color Mode: sRGB
- Max print area: 14" x 16"

### Screen Print
- Vector format: AI, EPS, or PDF
- Colors: Specified as Pantone or spot colors
- Trap: 0.5pt minimum for registration
